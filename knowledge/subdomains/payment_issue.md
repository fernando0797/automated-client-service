# Payment Issue

## Description
Problems encountered during payment processes, including failed transactions, incorrect charges, or billing errors.

## Common Issues
- Payment declined
- Duplicate charges
- Incorrect billing amount
- Payment not processed
- Subscription billing errors

## Root Causes
- Insufficient funds
- Bank restrictions
- Incorrect payment details
- Payment gateway errors
- Currency or region mismatch

## Troubleshooting Steps
1. Verify payment details
2. Check bank account status
3. Try alternative payment method
4. Contact bank if needed
5. Retry transaction
6. Check system status

## When to Escalate
- Multiple failed transactions
- Unauthorized charges
- Payment processed but service not activated

## Response Guidelines
- Be clear and transparent
- Avoid blaming the user
- Reassure about payment security
