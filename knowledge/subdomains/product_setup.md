## Description
Issues related to initial configuration and setup of a product.

## Common Issues
- Device not starting properly
- Setup process failing
- App pairing issues
- Configuration errors

## Root Causes
- Incorrect setup steps
- Missing dependencies
- Connectivity issues
- User misunderstanding

## Troubleshooting Steps
1. Follow official setup guide
2. Restart device
3. Ensure connectivity
4. Update firmware/software
5. Reset device if needed

## When to Escalate
- Setup fails repeatedly
- Device defective out of the box
- Critical onboarding failure

## Response Guidelines
- Provide step-by-step guidance
- Keep instructions simple
- Confirm each step
