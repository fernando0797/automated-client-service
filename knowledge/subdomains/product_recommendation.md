# Product Recommendation

## Description
Requests from users seeking advice on which product to choose based on their needs and preferences.

## Common Issues
- Choosing between multiple products
- Finding best value option
- Identifying suitable features
- Budget constraints

## Root Causes
- Lack of product knowledge
- Overwhelming options
- Specific use-case requirements

## Troubleshooting Steps
1. Ask about user needs
2. Identify budget range
3. Compare relevant products
4. Highlight key differences
5. Recommend best-fit option

## When to Escalate
- Highly specialized requirements
- Enterprise-level recommendations

## Response Guidelines
- Personalize recommendations
- Be neutral and unbiased
- Avoid upselling aggressively
