# Account Access

## Description
Issues related to users being unable to access their accounts. This includes login failures, password issues, account lockouts, and authentication errors.

## Common Issues
- Forgotten passwords
- Incorrect login credentials
- Account locked after multiple failed attempts
- Two-factor authentication (2FA) issues
- Expired sessions
- Unauthorized access alerts

## Root Causes
- User input errors (wrong password/email)
- Security policies (lockouts after failed attempts)
- Expired credentials
- Issues with authentication services
- Device/browser incompatibility
- Network-related issues

## Troubleshooting Steps
1. Verify email/username correctness
2. Reset password using official recovery flow
3. Check spam folder for reset email
4. Ensure correct 2FA method is used
5. Clear browser cache or try different browser
6. Verify system status (authentication services)
7. Check internet connection

## When to Escalate
- Account locked without user action
- Suspected hacking or unauthorized access
- Password reset not working
- Backend authentication failure

## Response Guidelines
- Be empathetic and reassuring (security is sensitive)
- Provide clear step-by-step recovery instructions
- Never ask for passwords
- Reinforce security best practices
