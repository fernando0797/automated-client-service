# Delivery Problem

## Metadata
```json
{
  "domain": "administrative",
  "subdomain": "delivery_problem",
  "type": "subdomain"
}
```

## Description
Issues related to shipping, delivery delays, lost packages, or incorrect deliveries.

## Common Issues
- Late delivery
- Package not received
- Wrong item delivered
- Damaged goods
- Tracking not updating

## Root Causes
- Logistics delays
- Incorrect address
- Courier issues
- Inventory errors
- External factors (weather, strikes)

## Troubleshooting Steps
1. Verify shipping address
2. Check tracking number
3. Contact courier service
4. Confirm delivery attempt
5. Offer replacement or refund if applicable

## When to Escalate
- Package marked delivered but not received
- High-value items lost
- Repeated delivery failures

## Response Guidelines
- Be proactive and reassuring
- Provide clear timelines
- Avoid blaming courier directly
- Offer solutions (refund/replacement)
