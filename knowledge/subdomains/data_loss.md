## Description
Situations where users lose files, data, or progress due to system errors, accidental deletion, or failures.

## Common Issues
- Deleted files
- Unsaved work lost
- Corrupted files
- Sync failures
- Backup not working

## Root Causes
- Software crashes
- User error (accidental deletion)
- Storage failures
- Sync conflicts
- Malware or system errors

## Troubleshooting Steps
1. Check recycle bin/trash
2. Restore from backups
3. Use version history if available
4. Check cloud sync status
5. Run recovery tools
6. Verify storage integrity

## When to Escalate
- Critical data loss (business impact)
- No backups available
- Corruption affecting multiple files
- System-level failures

## Response Guidelines
- Show empathy (high frustration scenario)
- Act quickly and clearly
- Avoid false guarantees of recovery
- Prioritize recovery steps first
