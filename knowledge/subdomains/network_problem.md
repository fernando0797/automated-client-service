## Description
Issues related to connectivity problems, including inability to connect to the internet, unstable connections, or network interruptions.

## Common Issues
- No internet connection
- Slow network speed
- Intermittent connectivity
- Wi-Fi not detected
- Connection drops frequently

## Root Causes
- Router/modem malfunction
- Weak signal strength
- ISP outages
- Incorrect network settings
- Firewall or security restrictions
- Device software issues

## Troubleshooting Steps
1. Restart router and device
2. Check Wi-Fi signal strength
3. Forget and reconnect to network
4. Reset network settings
5. Test with another network
6. Check ISP status
7. Disable VPN/firewall temporarily

## When to Escalate
- No connectivity across multiple devices
- Persistent ISP-related issues
- Hardware failure (router/modem)

## Response Guidelines
- Ask about scope (one device vs all devices)
- Provide simple step-by-step fixes
- Avoid technical jargon initially
