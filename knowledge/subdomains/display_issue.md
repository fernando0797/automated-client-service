# Display Issue

## Description
Problems related to screens, including visual defects, flickering, resolution issues, or display failures.

## Common Issues
- Screen flickering
- Black or blank screen
- Dead pixels
- Distorted colors
- Resolution problems

## Root Causes
- Hardware defects
- Driver issues
- Software bugs
- Loose connections
- Physical damage

## Troubleshooting Steps
1. Restart device
2. Update display drivers
3. Adjust resolution settings
4. Check external connections
5. Test with another display
6. Run diagnostic tools

## When to Escalate
- Physical damage
- Persistent hardware issues
- Screen completely non-functional

## Response Guidelines
- Ask about recent changes (drops, updates)
- Provide safe troubleshooting
- Avoid recommending hardware opening
