# Refund Request

## Metadata

```json
{
  "domain": "administrative",
  "subdomain": "refund_request",
  "type": "subdomain"
}
```

## Description
Requests from customers to receive a refund for a product or service.

## Common Issues
- Refund eligibility questions
- Delayed refunds
- Partial refunds
- Refund method issues

## Root Causes
- Dissatisfaction
- Billing errors
- Product defects
- Cancellation timing

## Troubleshooting Steps
1. Verify purchase details
2. Check refund policy
3. Confirm eligibility
4. Process refund request
5. Inform timeline

## When to Escalate
- Disputed refunds
- Policy exceptions
- High-value transactions

## Response Guidelines
- Be transparent about policies
- Show empathy
- Avoid conflict
