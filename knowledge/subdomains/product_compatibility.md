# Product Compatibility

## Description
Issues regarding whether different products, software, or systems can work together.

## Common Issues
- Software not compatible with device
- Integration failures
- Unsupported configurations
- Version incompatibility

## Root Causes
- System requirement mismatch
- OS version limitations
- Deprecated features
- Hardware limitations

## Troubleshooting Steps
1. Check compatibility specifications
2. Verify software version
3. Update system/software
4. Use supported configurations
5. Consult official compatibility lists

## When to Escalate
- Critical compatibility conflicts
- No available workaround
- Enterprise-level integration issues

## Response Guidelines
- Provide clear compatibility info
- Avoid vague answers
- Suggest alternatives when possible
