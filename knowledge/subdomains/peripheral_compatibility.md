# Peripheral Compatibility

## Description
Issues related to compatibility between a device and external peripherals such as controllers, headphones, keyboards, or accessories.

## Common Issues
- Device not recognizing peripheral
- Limited functionality
- Connection failures
- Unsupported accessory

## Root Causes
- Compatibility limitations
- Outdated drivers/firmware
- Incorrect connection method
- Hardware mismatch

## Troubleshooting Steps
1. Verify compatibility requirements
2. Update drivers/firmware
3. Reconnect device
4. Test on another device
5. Use official accessories if possible

## When to Escalate
- Peripheral not working despite compatibility
- Firmware-related issues
- Manufacturer defects

## Response Guidelines
- Clarify model compatibility
- Avoid assumptions
- Suggest official documentation if needed
