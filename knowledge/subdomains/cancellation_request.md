# Cancellation Request

## Description
Requests from customers to cancel subscriptions, orders, or services.

## Common Issues
- Subscription cancellation
- Order cancellation before shipment
- Service termination requests
- Accidental purchases

## Root Causes
- Change of mind
- Pricing concerns
- Dissatisfaction with service
- Duplicate orders
- Trial expiration

## Troubleshooting Steps
1. Verify user identity
2. Check order/subscription status
3. Confirm eligibility for cancellation
4. Inform about refund policy
5. Process cancellation request
6. Send confirmation to user

## When to Escalate
- Cancellation after shipment
- Complex billing disputes
- System errors preventing cancellation

## Response Guidelines
- Be polite and non-confrontational
- Offer alternatives if appropriate (pause, downgrade)
- Clearly explain refund implications
- Avoid retention pressure
