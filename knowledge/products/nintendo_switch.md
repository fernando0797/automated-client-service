# Nintendo Switch

## Overview
Nintendo Switch is a hybrid gaming console that can be used both as a handheld device and connected to a TV.

## Key Features
- Hybrid console (portable + docked)
- Detachable Joy-Con controllers
- Online multiplayer
- Game library

## Common Issues
- Joy-Con drift
- Connectivity issues
- Battery drain
- Dock not working

## Technical Specifications
- Storage: Internal + microSD
- Connectivity: Wi-Fi, Bluetooth
- Battery: Rechargeable

## Troubleshooting Guide
### Controller Issues
- Recalibrate Joy-Con
- Update firmware

### Dock Issues
- Check HDMI connection
- Restart console

### Battery Issues
- Charge fully
- Reduce brightness

## Compatibility
- Compatible with accessories
- Supports online services

## Best Practices
- Keep system updated
- Avoid physical damage
- Store safely
