# Lenovo ThinkPad

## Overview
Lenovo ThinkPad is a business-oriented laptop line known for durability, reliability, and strong performance in professional environments.

## Key Features
- Robust build quality
- Long battery life
- Business security features
- Comfortable keyboard with TrackPoint
- Enterprise support tools

## Common Issues
- Battery wear
- Keyboard issues
- Docking station problems
- Driver conflicts
- Boot errors

## Technical Specifications
- OS: Windows/Linux (model dependent)
- RAM: 8–32GB
- Storage: SSD
- Connectivity: Wi-Fi, Bluetooth, Ethernet

## Troubleshooting Guide

- Check BIOS settings
- Run system diagnostics

### Battery Issues
- Calibrate battery
- Replace if needed

### Docking Issues
- Update firmware
- Check connections

## Compatibility
