# HP Pavilion

## Overview
HP Pavilion is a consumer-oriented laptop series designed for everyday computing, including work, multimedia, and light gaming.

## Key Features
- Mid-range performance
- Lightweight design
- SSD storage options
- Integrated or dedicated graphics
- Windows OS

## Common Issues
- Slow performance
- Battery degradation
- Overheating
- Wi-Fi connectivity issues
- Driver conflicts

## Technical Specifications
- OS: Windows
- RAM: 8–16GB typical
- Storage: SSD/HDD
- Connectivity: Wi-Fi, Bluetooth

## Troubleshooting Guide
### Performance Issues
- Close background applications
- Upgrade RAM if possible
- Clean storage

### Overheating
- Clean vents
- Use cooling pad
- Adjust power settings

### Connectivity Issues
- Update network drivers
- Reset network settings

## Compatibility
- Compatible with standard peripherals
- Supports external monitors

## Best Practices
- Keep drivers updated
- Avoid overheating environments
- Perform regular maintenance
