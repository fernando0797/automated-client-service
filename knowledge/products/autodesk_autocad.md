## Overview
AutoCAD is a professional CAD software used for 2D and 3D design in engineering, architecture, and construction.

## Key Features
- 2D drafting and 3D modeling
- Precision drawing tools
- File compatibility (DWG, DXF)
- Cloud collaboration

## Common Issues

- Platform: Windows, macOS
- File formats: DWG, DXF
- Hardware requirements: High RAM and GPU recommended
